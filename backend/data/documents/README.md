# Documents Folder

Place your PDF files in this directory for the RAG knowledge base.

## How to use:
1. Drop any `.pdf` file into this folder
2. Commit and push to GitHub: `git add . && git commit -m "add documents" && git push`
3. Trigger a Render redeploy — the Knowledge Agent will automatically ingest the PDFs on startup

## Example documents you can add:
- Product manuals
- Research papers
- Company handbooks
- Technical documentation
